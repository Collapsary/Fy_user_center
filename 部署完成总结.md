## 🔧 环境变量配置

### Railway后端环境变量
```
SPRING_PROFILES_ACTIVE=prod
MYSQL_HOST=你的数据库主机
MYSQL_PORT=你的数据库端口
MYSQL_DATABASE=你的数据库名
MYSQL_USER=你的数据库用户名
MYSQL_PASSWORD=你的数据库密码
```

### Vercel前端环境变量
```
REACT_APP_API_URL=https://你的后端URL.railway.app
```
